# **İngilizce Modal ve Semi-Modal Yapıları Kapsamlı Kılavuzu**

Bu kılavuz, İngilizcedeki saf modalları (core modals), yarı-modalları (semi/phrasal modals), geçmiş zaman modallarını (perfect modals) ve tercih yapılarını sistematik bir şekilde yapılandırarak işlevleri ve Türkçe karşılıkları ile birlikte sunmaktadır.

## **1\. Saf Modallar (Core / Pure Modals)**

Asla ek almayan, özneye göre şekil değiştirmeyen ve arkalarından doğrudan fiilin yalın halini (V0) alan temel yapılardır.

| Modal Yapısı | İşlevi | Türkçe Karşılığı   |
| :---- | :---- | :---- |
| **can do** | Yetenek, izin, genel olasılık | yapabilir |
| **can't do** | Yeteneksizlik, yasaklama, imkansız güçlü tahmin | yapamaz |
| **could do** | Geçmiş yetenek, şimdiki zaman rica/olasılık | yapabilirdi, yapabiliyordu (eskiden), yapabilir |
| **couldn't do** | Geçmiş yeteneksizlik, imkansızlık | yapamazdı, yapamadı |
| **may do** | Olasılık (%50), resmi izin | yapabilir |
| **may not do** | Olumsuz olasılık, yasaklama | yapmayabilir, yapmamalı |
| **might do** | Zayıf olasılık | yapabilir |
| **must do** | Zorunluluk, güçlü olumlu tahmin (%95) | yapmalı, yapıyor olmalı |
| **mustn't do** | Yasaklama, güçlü negatif çıkarım | yapmamalı, olamaz |
| **shall do** | Gelecek zaman teklif/öneri (I & We ile) | yapacak, yapalım mı |
| **should do** | Tavsiye, beklenti | yapmalı, yapsa iyi olur |
| **shouldn't do** | Olumsuz tavsiye, yasaklama | yapmamalı, yapmasa iyi olur |
| **will do** | Gelecek zaman, söz, anlık karar, kibar rica | yapacak, yapar mısınız |
| **would do** | Geçmiş alışkanlık, koşul/gelecek ifadesi | yapardı, yapacaktı |

## **2\. Yarı-Modallar ve Phrasal Yapılar (Semi / Phrasal Modals)**

Anlam olarak modal işlevi gören ancak zamana, şahsa veya duruma göre (am/is/are/was/were/has) çekimlenen yapılardır.

| Modal Yapısı | İşlevi | Türkçe Karşılığı   |
| :---- | :---- | :---- |
| **is/are able to do** | Yetenek | yapabilir |
| **was/were able to do** | Zor şeylerin üstesinden gelme, geçmiş yetenek/başarı | yapabildi |
| **wasn't/weren't able to do** | Geçmişte üstesinden gelememe | yapamadı |
| **will have to do** | Gelecekte zorunda kalacak olma | yapmak zorunda kalacak |
| **have to / has to do** | Dışsal veya durumsal zorunluluk | yapmak zorunda |
| **had to do** | Geçmiş zorunluluk | yapmak zorunda kaldı |
| **have got to do** | Zorunluluk | yapmak zorunda |
| **don't/doesn't have to do** | Zorunluluk olmayışı | yapmak zorunda değil |
| **don't/doesn't need to do** | Gereklilik olmayışı | yapmak zorunda değil / gerek yok |
| **didn't have to do** | Geçmişte gereklilik olmayışı (Eylem yapılmadı) | yapmasına gerek kalmadı |
| **didn't need to do** | Geçmişte gereklilik olmayışı (Eylem yapılmadı) | yapmasına gerek kalmadı |
| **needn't do** | Gereklilik olmayışı | yapması gerekmez |
| **ought to do** | Tavsiye / beklenti | yapsa iyi olur / yapmalı |
| **ought not to do** | Tavsiye | yapmasa iyi olur |
| **had better (not) do** | Güçlü tavsiye (Olumsuz sonuç riski taşır) | yapmasa/yapsa iyi olur |
| **used to do** | Eski alışkanlık (Terk edilmiş) | eskiden yapardı |
| **didn't use to do** | Geçmiş alışkanlık olmayışı | eskiden yapmazdı |
| **is/are used to doing** | Alışkanlık durumu (Güncel) | yapmaya alışıktır |
| **was/were used to doing** | Geçmişteki alışkanlık durumu | yapmaya alışıktı |
| **get used to doing** | Alışma eylemi / süreci | alışmak |
| **is/are accustomed to doing** | Alışkanlık durumu | yapmaya alışıktır |
| **is/are to do** | Zorunluluk / Planlanmış eylem | yapmak zorunda / yapacak |
| **was/were to do** | Geçmiş planlama, azarlama/sitem | yapmalıydı/yapacaktı |
| **isn't/aren't supposed to do** | Yasaklama / Beklenti dışı durum | yapmaması gerekir |
| **was supposed to do** | Geçmiş beklenti / sitem | yapmalıydı/yapması gerekirdi |
| **be likely to do** | Tahmini muhtemel durumlar | muhtemel |
| **be bound to do** | Kaçınılmazlık | kaçınılmaz |
| **be obliged to do** | Zorunluluk | zorunlu |
| **be doomed to do** | Kötü sonuca mahkumiyet | mahkum |
| **be about to do** | Yakın gelecek eylemi | üzere |
| **be certain to do** | Kesinlik | kesin |
| **dare to do** | Cesaret etme | yapmaya cesaret eder |
| **daren't do** | Cesaret edememe | yapmaya cesaret edemez |
| **managed to do** | Zor şeylerin üstesinden gelme | yapabildi |

## **3\. Geçmiş Zaman Modalları (Perfect Modals)**

Geçmişe yönelik çıkarım, sitem, kaçırılmış fırsat veya pişmanlık bildiren yapılardır.

| Modal Yapısı | İşlevi | Türkçe Karşılığı   |
| :---- | :---- | :---- |
| **must have been doing** | Güçlü tahmin (Süreç) | yapıyor olmuş olmalı |
| **must have done** | Güçlü tahmin / çıkarım | yapmış olmalı |
| **must not have done** | Güçlü negatif tahmin | yapmamış olmalı |
| **can't have done** | Güçlü tahmin (Olumsuz) / imkansızlık | yapmış olamaz |
| **couldn't have done** | Güçlü tahmin / azarlama / sitem | yapmış olamaz / yapamazdı |
| **needn't have done** | Gerek kalmama (Ama yapıldı) | yapmasına gerek yoktu |
| **ought to have been doing** | Azarlama / fırça / sitem | yapıyor olmalıydı |
| **should be doing** | Tavsiye / beklenti | yapıyor olsa iyi olur / yapıyor olsa gerek |
| **should have been doing** | Azarlama / fırça / sitem | yapıyor olmalıydı |
| **should have done** | Azarlama / fırça / sitem / pişmanlık | yapmalıydı |
| **shouldn't have done** | Azarlama / fırça / sitem | yapmamalıydı |
| **will have been able to do** | Gelecekte bitmiş olacak yetenek | yapabilmiş olacak |
| **would have done** | Koşul / gelecek planı (Gerçekleşmemiş) | yapardı / yapacaktı |

## **4\. Tercih Yapıları (Preferences)**

Şimdiki veya geçmiş zamanda tercih durumlarını ifade eden kalıplardır.

| Modal Yapısı | İşlevi | Türkçe Karşılığı   |
| :---- | :---- | :---- |
| **would prefer to be doing** | Tercih (Şu anki süreçte) | yapıyor olmayı tercih eder |
| **would prefer to do** | Tercih | yapmayı tercih eder |
| **would prefer to have done** | Tercih / pişmanlık (Geçmiş) | yapmış olmayı tercih ederdi |
| **would rather be doing** | Şimdiki zamanda tercih (Süreç) | yapıyor olmayı tercih eder |
| **would rather do** | Şimdiki zamanda tercih | yapmayı tercih eder |
| **would rather have done** | Keşke / geçmiş tercih | yapmış olmayı tercih ederdi |
| **would rather not do** | Şimdiki zamanda olumsuz tercih | yapmamayı tercih eder |
